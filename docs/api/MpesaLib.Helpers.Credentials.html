Remarks

M-Pesa Core authenticates a transaction by decrypting the security credentials. Security credentials are generated by encrypting the base64 encoded initiator password with M-Pesa’s public key, a X509 certificate. The algorithm for generating security credentials is as follows:

  1. Write the unencrypted password into a byte array.
  2. Encrypt the array with the M-Pesa public key certificate.Use the RSA algorithm, and use PKCS #1.5 padding (not OAEP), and add the result to the encrypted stream.
  3. Convert the resulting encrypted byte array into a string using base64 encoding. The resulting base64 encoded string is the security credential.
